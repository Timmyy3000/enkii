# Support

## Questions / setup help

Open a [GitHub issue](https://github.com/Timmyy3000/enkii/issues/new/choose) with:

- workflow YAML
- relevant Actions run link
- logs (redact secrets)
- what you expected vs what happened

## Bug reports

Use the bug report issue template.
